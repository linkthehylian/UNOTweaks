# UNOTweaks

### Description
Tweaks for Ubisoft's "UNO" available on [Steam](https://store.steampowered.com/app/470220/UNO/).

### Features
- Skip intro videos
- Custom deck and card skins
- Toggle to enable or disable V-Sync
- Set your preferred FPS when V-Sync is disabled
